### 1. C언어의 소개와 프로그램 작성 방법
* 1.1 C언어란 무엇인가
	- C언어는 인간과 컴퓨터 사이의 의사소통을 위한 프로그래밍 언어
	- C언어는 운영체제 또는 운영체제 기반의 응용 프로그램을 개발할 때 사용하는 효율성이 높은 프로그래밍 언어

* 1.2 컴파일러란 무엇인가
	- 프로그래밍 언어로 작성된 인간의 의도를 기계가 이해할 수 있게 **기계어로 변환하는 변환기**
	- 기계어 : 기계가 이해하는 **2진 숫자(0과 1)**로 작성된 언어
	- UNIX 계열의 C 컴파일러에는 **GCC**가 존재, 윈도우즈 계열의 C컴파일러에는 **VISUAL C++**이 존재

* 1.3 프로그램 작성 방법 4단계
	- 프로그램 작성 단계 :  인간이 C언어의 문법에 맞게 **소스 코드 작성**
	- 컴파일 단계 : 소스파일을 기계가 이해할 수 있는 **기계어로 변환하는 컴파일 작업을 수행**하여 오프젝트 파일 생성(확장자는 .obj)
	- 링크 단계 : 링커는 **오프젝트 파일과 표준 라이브러리 함수를 연결하여 하나의 파일로 통합**하는 링크 단계를 수행하여 실행파일을 생성(확장자는 .exe)
	- 로딩과 실행 : 로더는 실행 파일을 주기억장치(RAM)에 로딩함으로써 CPU가 실행파일을 실행하도록 도와줌

* 1.4 C언어의 특징
	- C언어로 프로그래밍을 작성하면 다른 운영체제에서 거의 그대로 사용할 수 있음(**이식성이 뛰어남**)
	- C언어로 시스템의 **하드웨어 부분을 제어가능**
	- C언어는 순서를 지키는 **절차지향 언어**

### 2. C언어의 기본 구조와 표춘 입출력
* 2.1 C언어의 기본 구조
	- 전처리기와 헤더 파일
		1. 전처리기는 컴파일러가 **컴파일을 수행하기 전에 먼저 처리** 
		2. 헤더파일 : 헤더파일은 표준 라이브러리 함수들의 동작을 따로 정의해 놓은 파일을 의미
	- main() 함수
		1. main() 함수는 프로그램 실행 시 운영체제에 의해서 가장 먼저 호출되고 가장 나중에 종료됨

### 3. 변수란 무엇인가
* 3.1 변수 선언하기
	- 변수 : 데이터를 저장하는 **메모리 공간**
	- 변수 선언 방법 
		1. 정수형 변수 : 정수를 저장하는 변수(char, short, int, long)
		2. 실수형 변수 : 실수(소수점이 있는 값, 지수가 있는 값)를 저장하는 변수(float, double, long double)
	- 데이터 : **변수에 저장되는 값**
	- 데이터를 저장하기 위해서는 메모리 공간이 반드시 필요
	- 변수를 선언하면 반드시 변수 **초기화가 필요**

* 3.2 변수 선언 시 주의할 점
	- 변수 선언은 제일 앞쪽에 한다
	- C언어는 대소문자를 구분하므로 변수이름을 작성할 때 주의

* 3.3 변수의 시작 주소와 & 연산자
	- 변수가 메모리 공간의 어디에 있는지를 나타내는 것이 **변수의 시작 주소**
	- **&연산자**를 사용하면 변수의 시작 주소를 알 수 있음

### 4. 상수란 무엇인가
* 4.1 상수란
	- 상수 : 프로그램 실행 도중 값이 **변경되지 않는 데이터**

* 4.2 리터럴 상수와 심볼릭 상수
	- 리터럴 상수 : 글자 그대로 의미가 있어서 이름이 없는 상수
		1. 예 : **정수형 상수, 실수형 상수, 문자 상수, 문자열 상수**
		2. 문자열 상수 : 큰따옴표로 묶은 하나 이상의 문자로 구성된 문자열
			- 컴파일러는 큰따옴표로 묶은 문자열 상수를 저장할 때 문자열 상수 맨 마지막에 문자열의 끝을 알리는 종료 문자 \0을 자동으로 추가하는데 이것을 NULL문자라고 함
	- 심볼릭 상수 : 상수를 기호화하여 변수처럼 이름을 지어 쓰는 상수
		1. **const키워드**를 사용하여 심볼릭 상수 생성
		2. **매크로(#define문)**를 사용하여 심볼릭 상수 생성
